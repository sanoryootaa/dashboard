# Dashboard 仕様書

## 概要
iPhone向けスケジュール・タスク管理ダッシュボード。単一のindex.htmlで動作する。

## 絶対ルール

### 色
使っていい色: 白(#fff)、黒(#000)、グレー1色のみ(ライト:#f2f2f2/ダーク:#1a1a1c)、サブテキスト(ライト:#999/ダーク:#777)、カテゴリ5色、警告赤(#c82020)
これ以外のグレー禁止。ヘッダーの日付テキストは黒(var(--tx))にする。サブテキスト色はタスクのサブ情報のみに使う。

### カテゴリ5色
ライト: バイト#c89820, 大学#b85a28, 制作#3d8828, プライベート#1868a8, 活動#7038a0
ダーク: バイト#dab030, 大学#d07040, 制作#58a838, プライベート#3090d0, 活動#9858c0

### 警告・超過表示
- カード背景を赤くすることを禁止する（一切やらない）
- 超過した締め切りカード: カード全体にborder:2px solid var(--red)をつける（囲う）。左のカテゴリ色バーも維持
- 緊急（期限内だが閾値以内）: 通常表示のまま、バッジの色だけ赤に
- 超過バッジ: 日付表示の位置に「!超過」と赤文字で表示。カテゴリタグは別に必ず表示する（消さない）
- 各タスクに緊急しきい値スライダーあり（タスク詳細展開内、1-7日、デフォルト2日）。必ず実装すること

### カテゴリ表示（タグ）
濃い背景(var(--cc))+白文字。font-size:11px, font-weight:700, border-radius:6px, padding:3px 8px

### カテゴリ装飾（カード左端のバー）
左端の太いバーは廃止する。代わりにカテゴリ色のドット(8px丸)をカード内の左端に配置する。

### フォント
Zen Kaku Gothic New, 500(本文) / 700(見出し)
サイズ: 11, 13, 15, 17, 22, 28px

### 情報階層
ページタイトル: 28px/700
ヘッダー日付: 13px/500/黒（グレーにしない）
セクションタイトル: 17px/700
締め切りカウント数字: 22px/700/カテゴリ色
カード内タイトル: 15px/700
カード内日時: 13px/500/var(--sub)。時刻も消さずに表示する
タスクタイトル: 15px/500
タスクサブ: 11px/500/var(--sub)
タイトルと日時の間の余白はmargin-top:2pxに詰める

### 角丸
12px: カード、ボタン、入力。6px: タグ。16px: フィルターチップ

### ヘッダー
padding-topを44pxに減らす（上の余白が大きすぎる）。日付テキストはvar(--tx)（黒）にする。

### 下部ナビバー
padding-bottom: calc(8px + env(safe-area-inset-bottom))にして画面下のラウンドと余裕を持たせる

### フローティング入力バー
ナビバーの直上に常時表示。タップ時の動作:
1. 入力バーが画面上部に移動し、タイトル入力フィールドになる
2. その下にカテゴリチップ、日付ボタン、時刻入力、タスク/予定切替、緊急しきい値スライダーが表示される
3. さらにその下にキーボードが自動で出る（input要素にautofocusまたはJSでfocus()）
4. 背景オーバーレイタップで閉じる
5. 展開時のレイアウト（上から）: タイトル入力 → カテゴリ → 日付 → 時刻 → しきい値スライダー → タスク/予定切替 → 追加ボタン

### 禁止事項
- グレー2色以上使用
- box-shadow使用
- カード背景を赤くする
- 左端の太いカテゴリバー
- ヘッダー日付をグレーにする
- 締切時間を非表示にする
- ボタンを大きくしすぎる
- 余白を大きく取りすぎる

### アニメーション
- タスク完了: チェック→緑フラッシュ→0.3s待機→fadeout+slidedown
- 完了後の行詰め: 残ったタスクがふわっと上に詰まるアニメーション(transition: transform 0.2s ease)
- テーマ切替: body全体にtransition: background 0.3s, color 0.3sをつけてフェード
- タブ切替: crossfade(opacity 0.15s)
- 折りたたみ: max-height transition 0.2s ease
- 全ボタン: :active { opacity:.6 }

### テーマ切替
3ボタン常時表示。選択中:bg=var(--tx),color=var(--bg)。非選択:bg=var(--card),color=var(--tx)。

### スケジュールセクション
日付ストリップ: 横スクロール、各セル46px幅。今日:bg=var(--tx),color=var(--bg)。選択:bg=var(--card)。
タイムライン: 時刻表示+イベントカード。イベントカード内にカテゴリドット。
このセクションが操作不能にならないように注意。日付タップでタイムライン更新が動くこと。

### データ
localStorageに保存。タスク: {id,title,c,due,dt,done,sub:[],memo,ud:2}。日付はnew Date()。

### commitルール
変更後は git add . && git commit && git push
